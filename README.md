# 🍔 Swiggy Sales Analysis

This project analyzes **Swiggy food delivery sales data** to uncover patterns in customer ordering behavior, regional performance, and food preferences.

Using **Python data analysis tools**, the project performs data cleaning, KPI extraction, and visualization to generate insights that could help food delivery platforms improve **business strategy, marketing, and operational decisions**.

---

# 📊 Project Objectives

The main goals of this project are:

* Analyze **sales performance over time**
* Identify **daily, weekly, and monthly trends**
* Understand **customer food preferences**
* Identify **top-performing cities and states**
* Extract insights useful for **business decision making**

---

# 🛠 Technologies Used

* **Python**
* **Pandas**
* **NumPy**
* **Matplotlib**
* **Seaborn**
* **Jupyter Notebook**

These tools are used for **data preprocessing, statistical analysis, and visualization**.
